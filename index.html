Packages

I currently use GitHub/brettviren for all my publicly available software, documentation and configuration.

  • moo is a tool for schema driven code generation used in DUNE-DAQ and others.
  • zio provides a high-level interface to ZeroMQ, Zyre and implements some patterns described in the zguide. Provides the domo and flow protocols.
  • yamz is you and me, Zyre.
  • gegede is for general geometry description (for describing detector geometry).
  • herbie is a herbstluftwm interactive environment thingy.
  • rephile photo manager
  • cppzmq tour gives a walk through of one of the C++ bindings to ZeroMQ (and a wip companion giving a cppzmq version of ZeroMQ security "houses").
    • related generaldomo generalizes ZeroMQ majordomo to CLIENT/SERVER sockets (see also domo in ZIO)
  • zperfmq tests ZeroMQ performance, applied to a 100 Gbps test network.
  • gravio plays with graph visualization.
  • Two implementations of advanced-shell-history: fishql for Fish shell and shist for Bash shell.
  • nuosc++ package for calculating neutrino oscillation probabilities in vacuum, constant and arbitrary matter.
  • all my repos including many experimental dead ends and a few forked from others.
